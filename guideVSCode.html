📂 Ouvrir un Projet

Travailler avec des fichiers et dossiers

1

Ouvrir un dossier

Fichier → Ouvrir un dossier (ou glissez-déposez votre dossier dans VSCode)
Toujours ouvrir le dossier entier, pas juste un fichier !

2

Créer / modifier un fichier

Cliquez sur l'icône + dans l'explorateur pour créer un fichier.
Donnez-lui une extension : .html, .css, .js

3

Sauvegarder

Appuyez sur Ctrl + S pour sauvegarder.
Un point ● sur l'onglet = des modifications non sauvegardées !

Action Raccourci
Sauvegarder Ctrl + S
Annuler Ctrl + Z
Copier une ligne Shift + Alt + ↓
Chercher dans le fichier Ctrl + F

⬛ Le Terminal Intégré

Une console directement dans VSCode

1

Ouvrir le terminal

Terminal → Nouveau terminal
ou avec le raccourci Ctrl + ` (la touche accent grave)

2

Le terminal s'ouvre dans votre dossier

Vous êtes automatiquement dans le dossier de votre projet. Pas besoin de naviguer !

Commandes de base à connaître :

Terminal — bash
# Voir où vous êtes
$ pwd
/home/etudiant/mon-projet

# Lister les fichiers du dossier
$ ls
index.html style.css script.js

# Aller dans un dossier
$ cd mon-dossier
💡 Pourquoi le terminal ?
C'est ici que vous lancerez Git, installerez des outils, et exécuterez vos projets.
Incontournable pour tout développeur !

🆘 Problèmes Fréquents

Solutions rapides aux soucis les plus courants

❌ "Je ne vois pas mes fichiers !"
Solution :
→ Vous avez ouvert un fichier au lieu d'un dossier
→ Faites Fichier → Ouvrir un dossier et sélectionnez le bon dossier
❌ "Mon fichier a un point ● sur l'onglet"
Solution :
→ Vous avez des modifications non sauvegardées
→ Appuyez sur Ctrl + S pour sauvegarder
❌ "Le terminal ne s'ouvre pas"
Solution :
→ Essayez via le menu Terminal → Nouveau terminal
→ Ou vérifiez que vous avez bien ouvert un dossier (pas juste un fichier)
