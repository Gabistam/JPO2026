PHASE 1

🤖 L'IA Génère Tout le Code

Laissez ChatGPT créer la base complète de votre portfolio

1
Génération Automatique

Demandez à ChatGPT de créer l'intégralité de votre portfolio avec les 4 sections. L'IA va générer tout le code HTML/CSS nécessaire.

📝 Prompt à utiliser :

Crée un portfolio de développeur web avec les sections suivantes :
- Hero : mon nom et un slogan accrocheur
- About : une bio courte
- Projects : 3 cards de projets avec titre, description et image
- Contact : liens vers réseaux sociaux

Style moderne et minimaliste avec un design professionnel.

✅ Ce que vous allez obtenir :

  • Code HTML complet structuré en sections
  • CSS avec design moderne et responsive
  • Palette de couleurs harmonieuse
  • Mise en page professionnelle

💡 Conseil :

Copiez tout le code généré dans CodePen. Vérifiez que les 4 sections s'affichent correctement. Ne vous inquiétez pas si ce n'est pas parfait, vous allez personnaliser ensuite !

⏱️ Phase 1 : 10:00
PHASE 2

🧠 Compréhension du Code

L'IA vous explique comment fonctionne le code généré

2
Exploration & Apprentissage

Maintenant que vous avez le code, demandez à l'IA d'expliquer les parties importantes. C'est le moment de comprendre ce que vous avez créé !

📝 Questions à poser à ChatGPT :

  • À quoi sert cette balise <section> ?
  • Pourquoi utilise-t-on display: flex ici ?
  • Comment fonctionne le responsive dans ce code ?
  • Explique-moi les media queries ligne par ligne
Explique-moi ce code section par section :
- Que fait la section Hero ?
- Comment fonctionne la grille de projets ?
- Comment le site s'adapte au mobile ?

💡 Objectif de cette phase :

Vous devez comprendre la structure de votre portfolio. Cela vous aidera à le personnaliser efficacement dans les phases suivantes. N'hésitez pas à poser plein de questions !
