CODE BATTLE EXPRESS

Atelier Développement Web + IA

3 défis progressifs

12 min + 12 min + 11 min = 35 minutes

Votre assistant IA : ChatGPT, Claude ou Gemini

Votre outil : CodePen.io

Appuyez sur → pour commencer
DÉFI 1/3

💳 Ta Carte de Visite Web

Créez votre première page web personnalisée

✨ Objectifs :

Votre nom en GROS titre
Votre passion en sous-titre
Votre emoji préféré
Un fond coloré de votre choix

💡 Aide - Prompt pour votre IA :

Crée-moi le code HTML/CSS pour une carte de visite avec mon nom [TON NOM], ma passion [TA PASSION] et un fond [COULEUR]

📌 Remplacez les parties entre crochets par vos vraies informations !

⏱️ Temps : 12:00
DÉFI 2/3

🎨 Bouton Magique

Créez un bouton interactif avec des effets au survol

✨ Objectifs :

Un bouton qui change de couleur au survol
Une animation cool
Un message personnalisé sur le bouton

💡 Aide - Prompt pour votre IA (Étape 1) :

Crée un bouton HTML avec effet hover qui change de couleur au survol de la souris

🎯 Stratégie :

  • Demandez le code de base à votre IA
  • Testez-le dans CodePen
  • Demandez à l'IA de changer les couleurs
  • Demandez d'ajouter une animation différente
⏱️ Temps : 12:00
DÉFI 3/3 - FINALE

🏆 Battle Créative

Choisissez votre mission et créez le projet le plus stylé !

Choisissez votre mission :

🎮 Option A : Card de Jeu Vidéo

Image du jeu + Titre + Note /10 + Description

🎵 Option B : Player de Musique

Titre de chanson + Artiste + Bouton Play stylé

🎬 Option C : Affiche de Film

Titre + Tagline + Image + Note /10

🎯 Stratégie Gagnante :

  • Choisissez votre mission (A, B ou C)
  • Demandez à l'IA le code de base
  • Personnalisez au maximum (couleurs, images, textes)
  • Ajoutez des effets visuels cool (ombres, animations)
🏆 Le meilleur projet gagne un prix !
⏱️ Temps : 11:00
