🚀 Comment Démarrer

3 étapes ultra-simples

1

Ouvrir CodePen

Cliquez sur le lien qu'on vous donne
OU allez sur codepen.io/pen/

2

Coder dans les zones HTML et CSS

Tapez votre code et voyez le résultat apparaître automatiquement en bas !

3

Sauvegarder & Partager

Cliquez sur "Save" (en haut à droite) puis copiez l'URL pour partager votre création

💡 Pas besoin de compte pour aujourd'hui !

🆘 Problèmes Fréquents

Solutions rapides aux soucis les plus courants

❌ "Mon code ne s'affiche pas !"
Solutions :
→ Vérifiez qu'il n'y a pas d'erreur (balise mal fermée, point-virgule manquant...)
→ Cliquez sur "Run" en haut pour forcer le rafraîchissement
❌ "J'ai tout cassé !"
Solutions :
→ Appuyez sur Ctrl + Z plusieurs fois pour revenir en arrière
→ Ou rechargez la page (F5) et recommencez
❌ "Je ne vois qu'une zone de code !"
Solution :
→ Cliquez sur les onglets HTML / CSS / JS en haut de l'écran
